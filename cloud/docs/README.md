# yunlailai

**构建你的第二大脑**

yunlailai 是一个 AI Agent 记忆管理平台，帮助你的 Agent 跨设备、跨平台共享记忆、偏好、教训和技能。

## 核心功能

- 🧠 **记忆同步**：偏好、教训、技能自动跨设备同步
- 🤖 **多Agent支持**：OpenClaw、Hermes 等主流 Agent 共享同一份记忆
- 📊 **知识编译**：自动将零散记忆整理为结构化知识库
- 💡 **反思引擎**：Agent 从对话中自动提取偏好、教训，持续进化
- 🔒 **加密存储**：所有数据 AES-256-GCM 加密，隐私安全

# 📘 使用示例 (Usage Examples)

本文档展示了 `openclaw-plugin-self-growth` 在实际使用中的典型场景与预期效果。

## 💬 场景一：日常对话与持久化记录

当你在与 OpenClaw 进行日常沟通时，插件会在后台静默运行，将你们的原始对话按日期归档到 `chat_logs/` 目录中。

**用户输入：**
> “帮我写一个 Python 脚本，用来批量重命名当前文件夹下的所有图片。”

**OpenClaw 回复：**
> “没问题！这是一个基于 Python `os` 模块的批量重命名脚本，它会查找当前目录下所有的 `.jpg` 和 `.png` 文件并加上前缀...”

**🔍 后台实际发生：**
插件会自动在 `chat_logs/YYYY-MM-DD.md` 文件中追加一条带有时间戳的对话记录。这确保了你的每一次提问和 AI 的回答都被完整保留，方便日后回溯分析。

---

## 🧠 场景二：长期记忆的自动提取

当你多次向 OpenClaw 表达个人偏好或习惯时，插件会智能识别并将这些信息沉淀为长期记忆。

**用户输入：**
> “以后写代码的时候，变量名尽量用中文拼音缩写吧，这样我看起来更亲切。”

**OpenClaw 回复：**
> “好的，我已经记住了！在接下来的代码编写任务中，我会优先使用中文拼音缩写来命名变量。”

**🔍 后台实际发生：**
插件会将这条关键信息提取出来，写入 `memory/user_preferences.md` 文件中。下次当你开启新的对话窗口时，OpenClaw 依然能记得这个偏好，无需你重复提醒。

---

## 🛠️ 场景三：技能的自动生成与注册

当你发现 OpenClaw 在某个领域表现不佳，或者你希望它掌握一项新能力时，插件会根据复盘结果自动生成标准化的技能文件。

**用户输入：**
> “刚才那个接口调用的方式太繁琐了，你应该直接封装成一个简单的命令。”

**OpenClaw 回复：**
> “收到建议！我已经触发了自我进化机制，正在根据这次的经验生成一个新的技能模块...”

**🔍 后台实际发生：**
1. 插件会自动生成一个标准的 `SKILL.md` 文件，直接输出到 OpenClaw 默认工作区技能路径。
2. 该文件包含了新技能的触发条件、执行步骤和注意事项。
3. OpenClaw 会自动加载该技能，下次遇到类似需求时，它将直接调用这个优化后的标准化流程。

---

## ⏰ 场景四：每日自动复盘与异常修复

如果你在配置中开启了每日复盘功能（默认开启），插件会在每天固定时间（晚上 8 点）自动对当天的任务进行深度复盘，你不需要做任何操作。

**后台自动化流程：**
1. **扫描日志**：插件读取当天的 `chat_logs/`，寻找执行失败或用户纠正过的对话。
2. **生成报告**：如果发现异常（例如某条 Bash 命令报错），它会尝试分析原因并写入复盘报告。
3. **自我修复**：若启用了 `SELF_GROWTH_AUTO_OPTIMIZE_SKILLS=true`，插件会尝试给出修正后的指令或更新相关的技能规则，避免第二天犯同样的错误。

**你可以随时通过查看 `chat_logs/` 和 `memory/` 目录下的最新文件，来见证你的 OpenClaw 变得越来越聪明的过程！**

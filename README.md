# self-growth

让 AI Agent 具备自我成长能力的 OpenClaw 插件。

## 功能

- 🧠 **记忆系统** — 自动记录偏好、经验、教训，每日复盘生成结构化记忆
- 🎯 **长期目标追踪** — 识别并追踪用户的长期目标
- 📋 **技能管理** — 自动生成、优化、注入技能文件
- 💬 **对话日志** — 按天记录完整对话历史
- 🔄 **每日复盘** — 六阶段复盘流程，自动编译结构化记忆
- 🌐 **云同步** — 支持付费版云端备份（yulailai 云服务）
- 🎭 **人格注入** — 自动提取用户人格、沟通模式并注入上下文

## 安装

### 方式一：npm 安装（推荐）

```bash
npm install -g openclaw-plugin-self-growth
openclaw plugins install openclaw-plugin-self-growth
openclaw gateway restart
